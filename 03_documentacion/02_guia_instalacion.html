⚙️
Configurar ruta Python en VS Code
Abre Archivo → Preferencias → Configuración y busca nttdata.ibm para ajustar estas opciones:
ConfiguraciónValor por defectoDescripción
nttdata.ibm.pythonPath python Ruta al ejecutable Python. Ej: C:\Python311\python.exe
nttdata.ibm.toolsPath (auto) Directorio raíz que contiene ibm_twx_tools/
nttdata.ibm.defaultVersion 1.0.0 Versión que aparece en el banner y la documentación generada

Alias recomendados

Agrega estos alias para lanzar la suite más rápido desde cualquier directorio:

🪟 PowerShell
🍎 zsh
🐧 bash
$PROFILE (PowerShell)
# Abrir perfil de PowerShell notepad $PROFILE # Agregar estas líneas al archivo: Set-Alias twx "nttdata-ibm-twx" function twx-analyze($f) { nttdata-ibm-twx analyze $f } function twx-docs($f) { nttdata-ibm-twx docs $f } function twx-flows($f) { nttdata-ibm-twx flows $f } function twx-services($f) { nttdata-ibm-twx services $f } # Recargar perfil . $PROFILE
~/.zshrc
# Agregar alias a ~/.zshrc echo 'alias twx="nttdata-ibm-twx"' >> ~/.zshrc echo 'alias twx-docs="nttdata-ibm-twx docs"' >> ~/.zshrc echo 'alias twx-flows="nttdata-ibm-twx flows"' >> ~/.zshrc source ~/.zshrc # Uso: twx analyze mi_app.twx
~/.bashrc
echo 'alias twx="nttdata-ibm-twx"' >> ~/.bashrc source ~/.bashrc

Cheatsheet de comandos

Todos los comandos disponibles en CLI y en Copilot Chat.

CLI — Terminal

nttdata-ibm-twx analyze archivo.twx
Resumen JSON: nombre, versión, conteo de artefactos por tipo
nttdata-ibm-twx entities archivo.twx
Business Objects: campos, tipos, herencias, namespaces
nttdata-ibm-twx services archivo.twx
Servicios: pasos, inputs/outputs, dependencias
nttdata-ibm-twx flows archivo.twx
Diagramas Mermaid flowchart TD por proceso/servicio
nttdata-ibm-twx deps archivo.twx
Grafo de dependencias Mermaid entre artefactos
nttdata-ibm-twx docs archivo.twx -o reporte.md
Documentación Markdown completa con diagramas
nttdata-ibm-twx scripts archivo.twx
Scripts JavaScript/Groovy embebidos en servicios
nttdata-ibm-twx endpoints archivo.twx
URLs REST/SOAP referenciadas en integraciones
nttdata-ibm-twx entries archivo.twx
Artefactos que no son llamados por ningún otro (entry points)

GitHub Copilot Chat
