Capacidades de análisis
¿Qué extrae la suite de cada .TWX?
9 dimensiones de ingeniería inversa aplicadas automáticamente a cada artefacto IBM.
🏗️
Business Objects y modelos
Extrae cada BO con sus campos tipados, flags de lista/requerido, herencias entre tipos y namespaces IBM.
.bo campos tipos herencia
⚙️
Lógica de servicios
Mapea cada paso del servicio: tipo (Script, Human Task, Integration, Decision), nombre, inputs, outputs y estado de error.
.service pasos decisiones
📜
Scripts embebidos
Vuelca todo el código JavaScript / Groovy / server-side script contenido en pasos de servicio — listo para revisión y refactoring.
JavaScript tw.local server-script
🌐
Integraciones externas
Detecta todos los endpoints REST, SOAP y HTTP referenciados. Clave para identificar servicios externos que deben actualizarse en la migración.
REST SOAP WSDL URL
📊
Diagramas de flujo
Genera flowcharts Mermaid renderizables en GitHub, Confluence y VS Code — sin necesidad de IBM Designer instalado en ningún entorno.
Mermaid flowchart TD GitHub
🕸️
Grafo de dependencias
Mapea qué servicio llama a cuál, identifica artefactos huérfanos y entry points no documentados — esencial para planear el orden de migración.
graph LR llamadas entry points
