Arquitectura técnica
Módulos del generador Appian
Cada mapper es un módulo Python independiente que toma los JSONs de la Suite v1 y produce los XMLs de Appian. Copilot CLI asiste en las transformaciones complejas.
01 BusinessObjectMapper Nuevo
Convierte los Business Objects (12.*.xml) en Record Types y Data Types de Appian. Mapea campos, tipos de dato, herencia de clases y genera las relaciones entre objetos.
# bo_mapper.py from ibm_twx_tools import entity_extractor # Input: 02_entities.json # Output: records/{name}.recordType.xml # types/{name}.dataType.xml bo_mapper.convert(entities_json, appian_schema)
Input IBM BPM
02_entities.json
12.*.xml (twClass)
Output Appian
.recordType.xml
.dataType.xml
02 HHSInterfaceMapper Nuevo
Transforma los Human Services (HHS) en Interfaces SAIL de Appian. Los botones se toman de los Coach Views (64.*.xml) — regla crítica de la Metodología v3. Copilot genera el SAIL inicial.
# hhs_mapper.py # Input: 03_services.json (type=hhs) # 64.*.xml (coachViews → botones) # Output: interfaces/{name}.interface.xml # ⚠️ Botones SIEMPRE desde 64.*.xml hhs_mapper.convert(services, coach_views)
Input IBM BPM

Convierte los Integration Services (IS) en objetos de integración de Appian y genera los Connected Systems a partir de los endpoints REST/SOAP detectados automáticamente.
# is_mapper.py # Input: 03_services.json (type=is) # 07_endpoints.txt (URLs) # Output: integrations/{name}.integration.xml # connectedSystems/{url}.connectedSystem.xml is_mapper.convert(services, endpoints)

05 BPDProcessMapper Nuevo
El más complejo: convierte los BPDs en Process Models de Appian usando el orden real de sequenceFlow. Mapea los nodos (tareas humanas, servicios, gateways) a sus equivalentes en Appian Process Automation.
# bpd_mapper.py # Input: 06_flows.txt (Mermaid) # 25.*.xml (bpd, sequenceFlow) # Output: processes/{name}.process.xml # ⚠️ SIEMPRE sequenceFlow, nunca X/Y bpd_mapper.convert(flows, bpd_xmls)

06 JSExpressionMapper + Copilot IA requerida
La traducción de JavaScript IBM BPM a Appian Expression Language es la única transformación que requiere IA. Copilot CLI con el skill profuturo-twx traduce script por script y genera las Expression Rules.
# js_mapper.py + Copilot CLI # Input: 08_scripts.txt (455 KB JS) # Copilot: JS → Appian Expression Language # Output: expressionRules/{name}.rule.xml # Idioma: tw.local.X → local!X # tw.env.X → cons!X js_mapper.translate_with_copilot(scripts)

Formato de importación
Estructura del paquete appian-package.zip
Appian espera un ZIP con una estructura específica de carpetas y archivos XML. El generador construye exactamente este formato para que el import sea directo y sin errores.
appian-package.zip — estructura de importación Appian
appian-package/ ├── application.xml ← Manifiesto del paquete (nombre, versión, dependencias) │ ├── content/ │ ├── records/ ← Record Types (desde 12.*.xml Business Objects) │ │ ├── Cliente.recordType.xml │ │ ├── Solicitud.recordType.xml │ │ └── ... (54 Record Types) │ │ │ ├── dataTypes/ ← Data Types (estructura de los BOs) │ │ ├── ClienteType.dataType.xml │ │ └── ... (54 Data Types) │ │ │ ├── processes/ ← Process Models (desde 25.*.xml BPDs + servicios) │ │ ├── RedencionBono.process.xml │ │ ├── ValidacionUDI.process.xml │ │ └── ... (44 BPDs + 24 GSS) │ │ │ ├── interfaces/ ← Interfaces SAIL (desde HHS + Coach Views) │ │ ├── PantallaSolicitud.interface.xml │ │ ├── PantallaRevision.interface.xml │ │ └── ... (23 Interfaces HHS) │ │ │ ├── constants/ ← Constantes (desde 62.*.xml tw.env.*) │ │ ├── ID_SUBETAPA_DEV.constant.xml ← valor: 8627 │ │ ├── ID_SUBETAPA_PROD.constant.xml ← valor: 8628 │ │ └── ... (N constantes) │ │ │ ├── integrations/ ← Objetos de integración (desde IS) │ │ ├── IntegracionUDI.integration.xml │ │ └── ... (22 IS → integrations) │ │ │ ├── connectedSystems/ ← Connected Systems (desde endpoints) │ │ └── SistemaUDI.connectedSystem.xml │ │ │ └── expressionRules/ ← Expression Rules (JS → Appian, via Copilot) │ ├── calcularMonto.rule.xml │ └── ... (scripts traducidos) │ └── README.md ← Instrucciones de importación y notas de la migración
