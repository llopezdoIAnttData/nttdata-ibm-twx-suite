Skills disponibles
Qué usar y cuándo
Comando Para qué sirve Quién lo usa
/nttdata Menú launcher — muestra todos los skills disponibles. Punto de entrada si no sabes qué usar. Ambos
/analyze-Twx-Extract Pipeline completo sobre 1 TWX: extrae XMLs, corre los 9 ciclos y genera el HTML de Extracción Técnica. Paso 2 del pipeline. Carlos
/twx-multi-entities Análisis cruzado de N TWX: detecta BOs compartidos, clusters semánticos, candidatos a toolkit. Genera reporte HTML con score IA. Paso 3 del pipeline. Carlos
/twx-deps Grafo de dependencias: quién llama a quién entre artefactos. Esencial para Pavel para entender impacto de la reingeniería. Pavel
/twx-flows Diagramas Mermaid del orden real de pasos (por sequenceFlow). Pavel los usa para validar que la reingeniería respeta el flujo original. Pavel
/twx-entities Business Objects con campos y herencia. Carlos los usa para decidir qué va al toolkit de Appian. Carlos
/twx-services Servicios IS/GSS/HHS con pasos, inputs/outputs y tipo SYNC/ASYNC. Usado por ambos para entender la lógica antes del import. Ambos
/profuturo-twx Knowledge base completa de la metodología v3: 11 correcciones, patrones IBM→Appian, NOV_INSTANCE_MANAGEMENT, reglas de botones Coach Views. Consultar mientras se trabaja. Ambos
/twx-endpoints Endpoints REST/SOAP externos referenciados en el TWX. Carlos los necesita para los contratos de integración en Appian. Carlos
/nttdata-update Actualiza todos los skills desde GitHub. Ejecutar cuando haya nuevas versiones disponibles. Ambos
Instalación
Paso a paso — entorno completo
1
Verificar cuenta GitHub con Copilot activo
Necesitas una cuenta GitHub con licencia Copilot (Individual o Business). Confirmar con el responsable de licencias de NTT DATA EMEAL.
2
Instalar GitHub Copilot CLI
Seguir la guía de instalación completa en el documento 02_guia_instalacion.html de esta misma carpeta. Cubre VS Code + extensión + autenticación gh CLI.
3
Instalar Python 3.10+
Descargar desde python.org. Durante la instalación marcar ✅ "Add Python to PATH".
4
Clonar la suite NTT DATA IBM TWX
Solicitar acceso al repositorio al owner de la suite. Luego:
PowerShell
# Clonar el repositorio en Documentos cd "$env:USERPROFILE\Documents" git clone https://github.com/llopezdoIAnttData/nttdata-ibm-twx-suite.git NTTDATA-IBM-TWX-Suite
5
Instalar el paquete Python ibm_twx_tools
PowerShell
cd "$env:USERPROFILE\Documents\NTTDATA-IBM-TWX-Suite\01_herramientas_python" pip install -e . # Verificar instalación python -m ibm_twx_tools analyze --help
6
Instalar los skills NTT DATA en Copilot CLI
Abre Copilot CLI y ejecuta el skill de actualización:
Copilot CLI — chat
# En el chat de Copilot CLI escribe: /nttdata-update # Esto descarga e instala los 16 skills automáticamente # Al terminar verás: ✅ Instalación completada — 16 skills
7
Verificar que todo funciona
Copilot CLI — verificación
# Verificar skills instalados /nttdata # Debe mostrar el menú con las 14 opciones # Verificar Python python -m ibm_twx_tools analyze --help # Debe mostrar: usage: ibm_twx_tools analyze ...
✅ ¿Necesitas ayuda con la instalación? Contactar al owner de la suite. La guía detallada está en 02_guia_instalacion.html de esta misma carpeta.
