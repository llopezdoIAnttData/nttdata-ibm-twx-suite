# Soporte — NTTDATA IBM TWX Suite

## Antes de pedir ayuda

1. Lee el [README](README.md) — especialmente la sección **Inicio rápido**
2. Revisa los [Issues abiertos](../../issues) — puede que ya esté reportado
3. Consulta el [CHANGELOG](CHANGELOG.md) — tu versión puede tener un bug ya corregido

## Cómo obtener ayuda

### 🐛 Bug en el CLI Python o en un skill

Abre un [Bug Report](../../issues/new?template=bug_report.md) con:
- Tu versión de Python (`python --version`)
- Versión de la suite (ver `CHANGELOG.md`)
- El comando exacto que ejecutaste
- El mensaje de error completo

### 💡 Nueva funcionalidad o mejora

Abre un [Feature Request](../../issues/new?template=feature_request.md).

### ❓ Pregunta de uso / metodología IBM BPM → Appian

Abre una [Discussion](../../discussions) en la categoría **Q&A**. Las discusiones son para preguntas generales, no para bugs.

### 🔒 Vulnerabilidad de seguridad

Lee [SECURITY.md](SECURITY.md) y **no abras un issue público**. Escribe a **security@nttdata.com**.
